# Cat-Ling Advisory & Ethical Guidelines

This software is released under the MIT License to encourage freedom and innovation. However, the author (Cat-Ling) strongly requests that all users and developers adhere to the following ethical guidelines.

**These are not legal conditions of the license, but a request to use this software responsibly.**

### 1. No Data Collection
Please do not use this software to collect, store, sell, transmit, or analyze user data, telemetry, or "fingerprinting" data. This project was built to *protect* privacy, not invade it.

### 2. No Malicious Hijacking
Please do not use this software to forcefully or deceptively modify user search settings, browser homepages, or default search engines against the user's will. 
*   *Clarification*: Customizing search parameters for privacy (the intended use) is fine. Forcing those changes on unwilling users is not.

### 3. No Fraud
Please do not use this software for:
*   Search Arbitrage
*   Ad Fraud
*   Generating bot traffic
*   Impersonating other software or brands

We believe in an open web built on trust and privacy. Please help us keep it that way.
